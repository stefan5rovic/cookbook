# BC shortcuts
- Alt + Q - Tell Me
- Ctrl + Alt + F1 - page inspection
- F5 - refresh data on the page
- Ctrl + F1 - help for current page
- F8 - copy above
- F3 - search list
- Shift + F3 - filter
- Ctrl + Shift + Alt + F3 - reset filter
- F6 - next fast tab

## dates

 - t - today
 - w - work date
